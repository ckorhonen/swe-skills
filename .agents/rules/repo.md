# Repository Rules

Repository of `swe:` AI skills, eval fixtures, and the Cloudflare-hosted static skill site.

## Commands To Prefer

- `npm install`
- `npm run lint:md`
- `npm run evals:check`
- `npm run evals:packet`
- `npm run judges:check`
- `wrangler deploy`

## Repo-Specific Notes

- Every skill identifier/name must be prefixed with `swe:`.
- When skill behavior changes, check `site/index.html` and README accuracy.
