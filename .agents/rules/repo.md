# Repository Rules

Worktree of `swe-skills` for plugin metadata cleanup; uses the same skill, eval, and site layout as the main checkout.

## Commands To Prefer

- `npm install`
- `npm run lint:md`
- `npm run evals:check`
- `npm run evals:packet`
- `npm run judges:check`
- `wrangler deploy`

## Repo-Specific Notes

- Preserve the `swe:` skill naming convention.
- Coordinate with the main swe-skills checkout before duplicating broad metadata changes.
